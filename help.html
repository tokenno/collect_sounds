Sound Hunt

Welcome to Sound Hunt, an interactive GPS-based app designed to help you explore and discover hidden sounds around you! This manual will guide you through all the features and how to use them effectively.

Overview

Sound Hunt uses your device's GPS and compass to create an engaging experience where you can set or find target points, each associated with a unique sound (Sound 1, Sound 2, Sound 3, or Sound 4). The app supports two modes: Manual Mode for setting your own points and the default Admin Mode for discovering pre-set points.

Getting Started

  • Access the App: Open in a modern web browser on a device with GPS and compass capabilities (e.g., smartphone or tablet). Ensure you are on a secure HTTPS connection.
  • Grant Permissions: Allow the app to access your location and, if prompted, the compass when enabling it.
  • Initial Setup: The app will display "Waiting for GPS..." until it acquires your location. Once acquired, it will show your coordinates (e.g., "Lat: 12.3456, Lng: 78.9012").

Main Interface

The app is divided into sections visible on the screen:

  • Header: Displays the title "Sound Hunt" and a brief description.
  • GPS Status: Shows your current latitude and longitude or any GPS errors.
  • Message Area: Provides temporary feedback (e.g., success or error messages) that disappears after 5 seconds.
  • Debug Status: Offers technical logs for troubleshooting (visible to users for feedback).
  • Target Distance: Displays the distance to the selected target if a point is set and GPS is available.
  • Centered Section: Contains the "Start Tracking" button and the "Mixer" for sound control.
  • Container: Includes cards for Compass, Controls, Target Selection, Collected Sounds, and Manual Mode.

Key Features and Usage

  1. Start Tracking
    • Button: "Start Tracking" (turns red when active as "Stop Tracking").
    • Function: Toggles GPS tracking. Click to start, click again to stop.
    • How to Use:
      • Click "Start Tracking" to begin monitoring your position.
      • The app will update your location and check proximity to targets.
      • Stop tracking to pause all location-based actions.
  2. Mixer
    • Controls: Four sliders labeled "Sound 1," "Sound 2," "Sound 3," and "Sound 4" (range 0 to 1).
    • Function: Adjusts the volume of each sound independently.
    • How to Use:
      • Slide to change the volume (e.g., set to 0 for mute, 1 for full volume).
      • Affects sounds playing in both manual and tracking modes.
  3. Compass
    • Display: A circular canvas showing a red arrow indicating the direction to the selected target.
    • Button: "Enable Compass" (turns red when active as "Disable Compass").
    • Function: Provides directional guidance to the selected target using your device's compass.
    • How to Use:
      • Click "Enable Compass" to activate.
      • Select a target (see below), and the arrow will point toward it.
      • Disable to stop the compass update.
  4. Controls
    • Buttons:
      • "Reset All": Resets the app to its initial state.
      • "Sound Off": Stops all currently playing sounds.
    • How to Use:
      • Click "Reset All" to clear targets, sounds, and settings.
      • Click "Sound Off" to silence all active audio without resetting the app.
  5. Select Target
    • Buttons: Four buttons labeled "Target 1," "Target 2," "Target 3," and "Target 4".
    • Function: Allows you to choose a target to focus on, displaying its distance if set.
    • How to Use:
      • Click a target button to select it (it turns red when active).
      • If the target has a set point and GPS is available, the "Target Distance" area will show the distance (e.g., "Distance to Target 1: 12.34 meters").
      • Use with the compass for navigation.
  6. Collected Sounds
    • Display: Four lines showing the status of each sound (e.g., "Sound 1: Not found" or "Sound 1: Found").
    • Function: Tracks which sounds you have discovered.
    • How to Use:
      • Sounds update to "Found" when you enter within 5 meters of a target in tracking mode.
      • A message ("You found sound X") appears when a sound is collected.
  7. Manual Mode
    • Button: "Toggle Manual Mode" (turns red when active as "Disable Manual Mode").
    • Function: Enables you to set your own GPS points for targets.
    • How to Use:
      • Click "Toggle Manual Mode" to enable (disables admin points if set).
      • Four "Set GPS" buttons appear for Targets 1–4.
      • Click "Set GPS" for a target to set its location at your current position; the sound plays once to confirm.
      • Move outside 5 meters to make the point discoverable again in tracking mode.
      • Disable manual mode to revert to admin points.

How It Works

  • GPS Tracking: The app uses your device’s geolocation to monitor your position. Targets are considered "found" when you are within 5 meters.
  • Sound Playback:
    • In manual mode, setting a point plays the corresponding sound once.
    • In tracking mode, entering a 5-meter radius of a set point triggers the sound to play in a loop; it stops if you move beyond 30 meters.
    • A beep sound plays when you are between 5 and 30 meters from a target.
  • Discovery Eligibility: After setting a point manually, you must move outside 5 meters for it to become discoverable again.

Tips for Use

  • Best Experience: Use outdoors with a clear GPS signal and a calibrated compass.
  • Admin Points: If admin points are set (via localStorage or an admin page), they override manual points unless manual mode is active.
  • Troubleshooting: Check the "Debug Status" for logs or share them with support if issues arise.

Safety and Notes

  • Ensure you have permission to access locations where you set or find points.
  • Avoid using the app while driving or in unsafe areas.
  • The app requires internet access for audio file loading and HTTPS for geolocation.
